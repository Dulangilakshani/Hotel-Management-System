# 🏨 Hotel Management System

## 📌 Project Overview

The **Hotel Management System** is a group-based software project developed to automate and simplify hotel operations. The system helps manage guest information, room bookings, reservations, payments, and hotel services through an organized digital platform.

This project was developed collaboratively as a team, focusing on **software development practices, database management, user interface design, and teamwork**.

---

# 🎯 Objectives

* Automate hotel management operations.
* Manage guest and room information efficiently.
* Simplify the reservation process.
* Maintain accurate booking and payment records.
* Reduce manual work and improve operational efficiency.

---

# 🛠️ Technologies Used

* Java
* Java Swing (GUI)
* MySQL
* JDBC
* NetBeans IDE

---

# ✨ Features

### 👤 Guest Management

* Add new guest records
* Update guest information
* Delete guest details
* View guest records

### 🛏️ Room Management

* Add and manage room details
* Update room availability
* View available rooms
* Manage room categories

### 📅 Reservation Management

* Create room reservations
* Manage check-in and check-out details
* Update booking information
* Track reservation history

### 💳 Payment Management

* Record customer payments
* Manage billing information
* Maintain payment records

### 🗄️ Database Integration

* MySQL database connectivity using JDBC
* Efficient data storage and retrieval
* CRUD operations for system modules
